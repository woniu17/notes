### 字符串完整输出
```
set print elements 0
```

### gdb输出不分页
```
set pagination off
```

### 字符串输出
```
x/s char_array
x/10s char_array
```



### 打印函数参数
```
info args
```

### 打印函数局部变量
```
info locals
```

### gdb输出到文件中
```
set logging file <file name>
set logging on
print my_array
set logging off
```

### 打印结构体
```
set print pretty on/off
```

### 打印数组
```
set print array on/off
```
